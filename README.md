# 🛒 Blinkit Sales Analysis Dashboard (Power BI)

## 📊 Project Overview

This project presents a **Power BI dashboard** built to analyze Blinkit's sales performance, customer trends, and product insights.
The dashboard helps in understanding business growth and making data-driven decisions.

---

## *🚀 Features

* 📈 Sales performance analysis
* 🏬 Outlet-wise insights
* 📦 Product category distribution
* 👥 Customer behavior trends
* 💰 Revenue and profit visualization
